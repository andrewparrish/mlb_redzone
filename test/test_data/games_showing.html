Media Player Accessibility Information

Are you trying to access the Media Player using a screen reader?

  • The Media Player application found on this page is an accessible Flash application and allows subscribers of Gameday Audio to access game audio broadcasts and subscribers of MLB.TV to access game video and audio broadcasts.
  • To access this Media Player, you must be an subscriber. Existing subscribers should log in. New users must subscribe to access this Media Player.
  • Additionally, subscribers can use our standalone and accessible version of Gameday Audio that uses less computer resources.
  • This media player has been optimized for supports JAWS version 16 and Mac OSX Voice Over
  • If you are having trouble accessing this Media Player please visit the Accessibility FAQs or the Media Player support forum. Additionally you can contact our accessibility dedicated Customer Service at access@website.mlb.com or via phone at 866.239.1284 (international customers call 512.434.1542)
